ขั้นแรก สร้างโฟลเดอร์งาน และ nmp install prompt-sync ก่อนค่ะ
แล้วไปสร้าง repo เอา folder งานขึ้น git ไว้ก่อน

ลองเทสการเรียกใช้ prompt-sync ก่อน เพื่อให้แน่ใจว่าสามารถรับคำสั่งจากผู้ใช้ได้

============

ก่อนจะคิดโค้ด เราจะคิดแบบคนเล่นเกมก่อน:

เกมนี้คือให้เราเดินหาหมวก ระหว่างทางมีหลุม ห้ามตก แล้วก็ห้ามเดินออกนอกแผนที่
แอบคิดถึงพวกเกม pacman หรือเกม ระเบิด สมัยก่อนอยู่เมือนกันค่ะ น่าจะคล้ายๆกัน ว่าแล้วก็อยากเล่น :P

แปลว่าเกมนี้มี:

1. ตัวเรา
2. จุดหมาย (หมวก)
3. อันตราย (หลุม)
4. พื้นที่เดิน (map)

หลังจากเข้าใจว่าเกมมีองค์ประกอบอะไรบ้าง ก็คิดต่อว่าอะไรที่ต้องเก็บข้อมูลไว้ระหว่างเกม
เช่น ตำแหน่งผู้เล่นที่ต้องเปลี่ยนตลอดเวลา และแผนที่ที่ใช้ตรวจสอบว่าผู้เล่นเจออะไร เพราะเราจะต้องใช้ตัวแปรเพื่อเก็บข้อมูลเหล่านี้

แต่ก่อนจะสร้างตัวละคร ต้องสร้างแผนที่ก่อน ยังไม่ต้องเอาตัวละครไปวาง
เริ่มจากสร้างแผนที่ของเกมก่อน โดยเลือกใช้ array 2 มิติ เพราะสามารถแทนตารางได้
